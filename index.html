SISTEM DETEKSI ERROR

Pilih Metode Simulasi

Pilih metode untuk memproses data pada sisi pengirim dan memeriksa data pada sisi penerima. Panjang data dapat menyesuaikan parameter metode.

Σ

Checksum

Penjumlahan komplemen satu dengan end-around carry untuk mendeteksi perubahan data.

÷

CRC

Pembagian modulo-2 dengan XOR untuk memperoleh FCS dan memeriksa remainder Rx.

H

Hamming Code

Bit paritas dinamis untuk mendeteksi dan mengoreksi satu kesalahan bit.
